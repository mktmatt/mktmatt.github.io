Palm Desktop by ACCESS 6.2 for Windows

發表於 2008-02-20 分類於 windows
文章字數： 1.4k 所需閱讀時間 ≈ 3 分鐘

Windows Vista ，好說歹說，也問世一年多，時間過得真快，就某些部分對我來說，過得真慢…

期間，諸多透過小窗子所使用的軟體，業已隨之更新，軟體使用量偏低的我，確實較不受影響，偏偏手頭上的 Treo 650 要在 Windows Vista 上頭同步困難重重，懶得找解決方法，也不想用破破的 Beta 版 (( 一般來說，使用 Beta 版的軟體，風險自負。 Beta 也被 Google 廣泛使用，惟意境不同。 )) ，無奈地坐以待之。偶爾，我會去官方網站晃晃，要是剌眼的 Beta 字樣仍在，便毫不猶豫轉台。如此情狀，直到昨晚才告終結，邂逅之初我還特別把眼睛睜大點確認哩！接著，我開始閱讀五要點的 What’s new? (( 以往看到軟體就抓下來，橫衝直撞的測試，常當白鼠而吃鱉，慢慢學乖嚕！ )) ：

  1. Integrated Desktop：支援的 Palm OS 版本自 3.5.x 至 5.4.x。
  2. Enterprise Friendly HotSync Enhancements：HotSync Manager 7.0 成為獨立程式，不需要再與 Palm Desktop 綁在一起。
  3. Multilingual Support：巴西式葡萄牙語 (( 我對 Brazilian Portuguese 了解有限，若有翻譯上的錯誤，煩請告知，感謝。 )) 、英語、法語、義語、德語與西班牙語，沒有繁體中文，我有點失望。
  4. Install Tool：取代原本的 Palm Quick Install ，為軟體安裝與資料庫更新的同步工具。
  5. Outlook 2007 synchronization：以 USB Cable (( 只能用 USB Cable 嗎？可惜的是我目前也無法拿 Bluetooth 測試。 )) 讓 Palm 與 Outlook 2007 同步。

最後，我很肯定不會安裝 Palm Desktop ， HotSync Manager 與 Install Tool 就能夠滿足我的需求，說實在的老早之前我就不喜歡 Palm Desktop 與 HotSync Manager 感情太好。可惜的是 Palm Desktop by ACCESS 6.2 for Windows 不支援 64 位元的系統與搭載 2 顆以上 CPU 的硬體 (( 雙核或多核心處理器不受影響。 ))，想見更新版本可能會因應 64 位元的系統普遍後問世。未來， Palm 怎麼走下去，我個人也有深深的不確定感，轉換到 Windows Mobile (( iPhone 仍是我最期待的，唉！好像又提到另一個讓人等很久的實際案例。 )) 的可能絕對存在，能夠只安裝 HotSync Manager ，讓事情變得單純是再好不過。
